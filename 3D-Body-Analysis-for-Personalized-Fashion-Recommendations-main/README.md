📄 Project Details

SmartFit is an AI-based styling system that analyzes human body proportions using computer vision. The system processes front and side full-body images to extract key body landmarks, evaluate body structure, and generate personalized formal clothing recommendations.

It performs both front-view proportion analysis (shoulder width, hip width, body ratios) and side-view analysis (posture, body depth, and alignment). Based on these insights, the system classifies body type and suggests appropriate formal wear including blazer fit, trouser fit, shoulder padding, and jacket length.

The project focuses on bridging computer vision with fashion by providing data-driven styling guidance.

🛠️ Tech Stack
Programming Language: Python
Computer Vision: OpenCV
Pose Detection: MediaPipe
Numerical Computation: NumPy
Frontend/UI: Streamlit
Development Tools: VS Code
